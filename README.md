# Dhenz14 Pages host

This public repository contains generated static website bytes only. The
`HivePoA/` directory is an audited, non-authoritative frontend mirror generated
from the private canonical `Dhenz14/HivePoA` repository.

It is not HivePoA source, a tester package channel, a signed bootstrap handoff,
or storage-release authority. HivePoA tester packages and the package-bound
storage-control web interface remain signed-IPFS-only. The obsolete public
`spknetworkpoa` repository must never be used as current source or distribution.
